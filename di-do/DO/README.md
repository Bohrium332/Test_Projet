# DO接口测试脚本操作说明书

## 目的
此脚本用于测试 Jetson 开发板上的数字输出（DO）接口，控制 DO1（106 引脚）和 DO2（43 引脚）的高低电平状态。低电平时输出最大 30V 电压，高电平时输出 0V。脚本通过每隔 200 毫秒翻转一次引脚状态，进行连续的测试。

## 环境要求
1. 必须具有 `sudo` 权限以操作 GPIO 引脚。
2. 请确保安装gpiod库 `sudo apt install gpiod`。

## 使用说明

### 1. 脚本功能
- **DO1（106引脚）** 和 **DO2（43引脚）** 控制：
  - 当引脚为低电平时，DO 输出最大 30V 电压。
  - 当引脚为高电平时，DO 输出 0V 电压。
- 每隔 200 毫秒，脚本会将引脚状态从低电平切换到高电平，反复循环。

1. **下载脚本**：
   将脚本文件保存为 `DO_two_toggle.sh`。
